# aim_trainer_
🎯 Aim Trainer Game (Python)

An interactive aim training game built using Python and Pygame to improve mouse accuracy and reaction speed. The game challenges users to click on randomly appearing targets within a limited time while tracking their performance through score, misses, and reaction time.

🚀 Features
🎯 Randomly appearing targets
⏱️ 30-second gameplay timer
📊 Real-time score tracking
❌ Miss counter
⚡ Average reaction time calculation
🧾 Game over summary screen

🛠️ Tech Stack
Python
Pygame

🎮 How to Play
Click on the red targets as quickly as possible
Each hit increases your score
Missing a target increases miss count
Try to maximize your score within 30 seconds

▶️ Installation & Run
pip install pygame
python aim_trainer.py
